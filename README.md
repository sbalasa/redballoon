# redballoon
## Decipher UDP packets

Refer README.pdf for more info

### Steps to run:

- Install required packages
    <code>
      $ pip install -r requirements.txt
    </code>

- Client:
   <code>
      $ python send.py
   </code>

- Server:
   <code>
      $ python server.py --help
   </code>

- Verify Logs
   <code>
      $ python verify.py
   </code>
